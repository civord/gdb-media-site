# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability affecting this site, please report it directly to:

**info@gdbmedia.com.au**

Please include a clear description of the issue, the steps to reproduce it, and any relevant details that will help us understand and resolve it quickly.

We aim to acknowledge reports within 2 business days and will keep you updated as we investigate and address the issue. Please do not publicly disclose the vulnerability until we have had a chance to review and fix it.
